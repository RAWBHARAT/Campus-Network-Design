# Smart Campus Network Design using Cisco Packet Tracer

## 📖 Overview

This project demonstrates the design and implementation of a Smart Campus Network using Cisco Packet Tracer. The network is divided into multiple VLANs representing different departments to ensure efficient traffic management, security, and scalability. DHCP services are configured for automatic IP allocation, while Inter-VLAN Routing enables communication between departments.

The project simulates a real-world enterprise campus network and focuses on network segmentation, connectivity, performance, and security.

## ⚙️ Features Implemented

### VLAN Segmentation

* Department-wise network separation
* Improved security and traffic management

### DHCP Configuration

* Automatic IP address assignment
* Dynamic gateway allocation

### Inter-VLAN Routing

* Communication between different departments
* Efficient routing implementation

### Trunk Configuration

* IEEE 802.1Q trunking
* Multiple VLAN traffic over a single link

### Network Security

* Basic ACL implementation
* Controlled access between network segments

### Layer 3 Switching

* Faster routing and traffic forwarding

### Spanning Tree Protocol (STP)

* Prevention of switching loops
* Improved network stability
